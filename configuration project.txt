安装node.js

安装TypeScript 
cmd: npm install -g typescript 

安装@Angular/cli
cmd: npm install -g @angular/cli

安装vscode

安装vscode chrome调试插件 : Debugger for Chrome

安装apache(或任意的http server)
在httpd.conf文件里
1:将DocumentRoot定位到工程根目录
2:<Directory />节点下
  Require all denied
  Order deny,allow
  Deny from all
  如果有这几项,注释或删掉
